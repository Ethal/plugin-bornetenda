== Präsentation ==
Dieses Plugin erlaubt es, die Anschlüsse von Tenda W301A zu verwalten.

Es erlaubt, das WLAN zu aktivieren und deaktivieren.

== Installation/Einrichtung ==
Wir werden jetzt ein Gerät einrichten. Um dort hin zu kommen, klicken Sie auf '"Plugins / Kommunikation / tenda W301A'" 

Dann auf die Schaltfläche oben links "'''Gerät hinzufügen'''" klicken

Dann den Namen des Gerätes eingeben (z.B. Gerät 1)

Dann definieren :

* '''Eltern-Objekt'''
* ""Kategorie"' (optional)
* '''Aktivieren''' (ankreuzen, andernfalls wird das Gerät nicht nutzbar sein)
* '''Sichtbar''' (optional, wenn Sie nicht wünschen, dass es auf dem Armaturenbrett sichtbar ist)
* '''IP Adresse'''
* '''Konto'''
* '''Passwort'''
