# Plugin Borne Tenda W301A pour Jeedom

![Icone](plugin_info/bornetenda_icon.png)

### Fonctions disponibles

Infos :
* Status de la borne
* Status du Wifi

Actions :
* Activer le Wifi
* Désactiver le Wifi

Scénarios possible (ceux que j'ai chez moi) :
* Activation/désactivation du Wifi sur plage horaire
